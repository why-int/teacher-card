# Teacher Card Builder (Windows + macOS)

Готовое десктоп-приложение с красивым GUI:

- загрузка файла РУП (`.xlsx/.xlsm/.xls`)
- выбор одного или нескольких преподавателей
- редактирование данных по каждому преподавателю в таблице
- переключение между преподавателями без потери правок
- создание карточек по шаблону и сохранение в выбранный файл

## Быстрый старт (разработка)

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

На Windows:

```bat
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Сборка standalone-приложения

Важно: собирать нужно на целевой ОС.

- `.exe` собирается на Windows
- `.app` собирается на macOS

### Windows (.exe)

```bat
run\build_windows.bat
```

Результат: `dist\ExcelDataViewer.exe`  
Его можно запускать двойным кликом (иконка в Проводнике/на рабочем столе).

### macOS (.app)

```bash
chmod +x run/build_macos.sh
./run/build_macos.sh
```

Результат: `dist/ExcelDataViewer.app`  
Запуск как обычного приложения через Finder/Launchpad.

## Иконка приложения

Чтобы задать свою иконку, добавьте в папку `assets`:

- `app_icon.ico` для Windows
- `app_icon.icns` для macOS
- или `app_icon.png`

## Сценарий работы

1. Нажмите `Загрузить РУП` и выберите файл РУП.
2. В окне выбора отметьте нужных преподавателей.
3. В выпадающем списке переключайте преподавателей и при необходимости редактируйте их строки.
4. Нажмите `Создать карточки` и выберите место сохранения результата (шаблон берётся автоматически из проекта: `Пример.xlsx`, либо `Шаблон и пример.xlsm`).

Учебный год в шапке карточки берётся из имени файла РУП (формат `2024-2025`, `2025-2026` и т.д.).
